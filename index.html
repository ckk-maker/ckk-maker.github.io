FastAPI项目启动

发表于 2025-12-21

问题

Fast API项目，在Pycharm中启动，终端显示Error: Missing argument 'APP'.，项目无法正常运行，无法访问

解决

检查运行/调试配置，几个关键的参数如下：

  • python脚本——项目默认的，也就是当前使用的虚拟环境根目录下的python.exe
  • script脚本——当前虚拟环境下的uvicorn.exe：D:/conda_envs/calendar_task/Scripts/uvicorn.exe
  • 脚本参数——main:app --reload
  • 工作目录——一般就是项目的根目录

此次问题是因为脚本参数没有配置，导致uvicorn.exe启动后无法找到app = FastAPI()，继而无法正常运行。

Next数组

采用右移一位的格式

作用：当模式串在匹配文本串的过程中出错时，给出模式串下次匹配开始的下标（从0开始）。

每个字符对应的值就是该字符的前面所有字符的最长相同前后缀

最长相同前后缀：一个字符串的前k个字符为前缀，后k个字符为后缀，若前缀和后缀相同，且k小于字符串的长度n，那么最大的k就是最长相同前后缀。

模式串如下：

a a a b a a a f
0 1 2 0 1 2 3 0 默认
-1 0 1 2 0 1 2 3 右移

第一个字符a：默认是-1

第二个字符a：前面所有字符为”a”，故不存在最长相同前后缀，长度是0；

第三个字符a：前面所有字符为”aa”，故最长相同前后缀为”a”，长度是1；

第四个字符b：前面所有字符为”aaa”，故最长相同前后缀为”aa”，长度是2；
